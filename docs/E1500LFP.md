# Pecron E1500LFP — Portable Power Station

The E1500LFP is a **1,536Wh LiFePO4 portable power station** with AC and DC outputs. It's Pecron's mid-range unit — solid for home backup, camping, and workshop use.

## Specs

- **Capacity:** 1,536Wh (LiFePO4)
- **AC Output:** 1,500W continuous (pure sine wave), 120V/60Hz default
- **DC Output:** USB-A, USB-C, 12V barrel (center positive ⊕)
- **DC Input:** Solar/car up to ~500W
- **AC Input:** Wall charging
- **Expansion:** Supports expansion battery packs

> **Note:** The DC barrel plug is **center positive** (tip = +, ring = −). Verified with multimeter.

## Sensors

| Sensor | Description | Unit |
|--------|-------------|------|
| Battery % | State of charge | % |
| Voltage | Pack voltage | V |
| Current | Charge/discharge current | A |
| Temperature | Host pack temperature | °C |
| Total Input Power | Combined AC + DC input | W |
| Total Output Power | Combined AC + DC output | W |
| AC Input Power | Wall charger input | W |
| DC Input Power | Solar/car input | W |
| AC Output Power | AC inverter output | W |
| AC Output Voltage | Inverter output voltage | V |
| DC Output Power | 12V/USB output | W |
| Device Status | Shut Down / Charging / DC Discharge / AC Discharge / Standby / Protection | — |
| Remaining Time | Estimated runtime at current draw | H:M |
| Remaining Charging Time | Time until full | H:M |

## Configurable Settings

| Setting | Code | Options |
|---------|------|---------|
| AC Output | `ac_switch_hm` | On / Off |
| DC Output | `dc_switch_hm` | On / Off |
| UPS Mode | `ups_status_hm` | On / Off |
| AC Output Voltage | `ac_output_voltage_io` | 100V, 110V, 120V, 220V, 230V, 240V |
| AC Output Frequency | `ac_output_frequency_io` | 50Hz, 60Hz |
| Screen Brightness | `machine_screen_light_as` | 5%, 20%, 50%, 80%, 100% |
| Auto-Dim on Idle | `auto_light_flag_as` | On / Off |
| No-Output Auto-Off | `noastime_io` | Off, 1h, 2h, 3h, 4h |

## Data Sources

The E1500LFP reports **full telemetry via both local TCP and MQTT cloud**, so you get complete data regardless of connection method. This is in contrast to the E3800LFP which only sends settings via local TCP.

### Local TCP
Full data: battery %, voltage, current, temperature, all power readings, switch states, and charging pack info.

### MQTT Cloud
Same data as local TCP, delivered in real-time via Pecron's cloud MQTT broker.

## Home Assistant Integration

All sensors and switches above are auto-discovered when running with `--homeassistant`. The E1500 is the most straightforward model — no partial packet merging needed, everything comes in one clean payload.

## Charging Pack Data

The E1500 reports detailed charging pack information (`charging_pack_data_jdb`) including per-pack voltage, current, temperature, and status. Up to 4 expansion packs are supported.

## Example Commands

```bash
# Turn AC output on
python3 pecron_monitor.py --ac on

# Turn DC output off  
python3 pecron_monitor.py --dc off

# Set AC voltage to 120V (North America)
python3 pecron_monitor.py --control ac_output_voltage_io 2

# Set screen brightness to 50%
python3 pecron_monitor.py --control machine_screen_light_as 2

# Set auto-off to 2 hours when no load detected
python3 pecron_monitor.py --control noastime_io 2
```
